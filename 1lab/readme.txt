1) Спросить имя → вывести привет с f-строкой - what_is_name.py
2) Ввести два числа → вывести сумму, среднее, частное и остаток и проверить каждое на четность - twonumb.py
3) Ввести строку → развернуть и посчитать количество слов - strok.py
4) Напишите программу, которая выводит все чётные числа от 1 до 10 - chet.py
5) Программа для подсчета факториала введенного числа - facktorial.py
6) Найти N-ое число фибоначчи - fibanachi.py

При итерировании нужно использовать модуль tqdm


Установка зависимостей:


pip install -r requirements.txt
